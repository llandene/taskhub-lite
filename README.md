# TaskHub Lite

TaskHub Lite is a small modular task management system built to practice GitHub workflow and software project organization.

## Project structure

- `frontend/` - user interface
- `backend/` - API and business logic
- `shared/` - shared schemas and constants
- `docs/` - technical documentation
- `scripts/` - helper scripts
- `tests/` - automated tests

## Initial goal

Build a simple app that can:

- create tasks
- list tasks
- mark tasks as done
- delete tasks
