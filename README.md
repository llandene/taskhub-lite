# TaskHub Lite

TaskHub Lite is a small modular task management application built with Python, Flask, HTML, CSS, and JavaScript.

## Features

- list tasks
- create tasks
- complete tasks
- delete tasks

## Project structure

- `frontend/` - user interface
- `backend/` - Flask API and application logic
- `shared/` - shared task schema
- `docs/` - technical documentation
- `scripts/` - helper scripts
- `tests/` - automated tests

## Requirements

- Python 3
- virtual environment
- dependencies from `requirements.txt`

## Install

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
